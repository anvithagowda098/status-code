# Status Code Script

This project reads a list of URLs from a CSV file and prints their status codes using the Python requests library.
It essentially prints the status code if the URL is accessible else prints the specific error associated with the request to help debug why a link failed.

Few other key points of the script:
- Automatically skips header rows, empty lines and non url text.
- Uses utf-8-sig encoding to support CSV files generated by Microsoft Excel.

## Quick start
1. pip install requests
2. python check_url.py
